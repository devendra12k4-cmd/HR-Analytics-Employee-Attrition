# HR Analytics Employee Attrition Analysis

## Project Overview

This project analyzes employee attrition using the IBM HR Analytics dataset.

The objective is to identify factors contributing to employee turnover and evaluate the dataset's readiness for machine learning.

---

## Dataset

IBM HR Analytics Employee Attrition Dataset

Records: 1470 Employees

Features: 35 Variables

Target Variable: Attrition

---

## Project Workflow

1. Data Quality Assessment
2. Exploratory Data Analysis (EDA)
3. Outlier Detection
4. Outlier Treatment
5. Feature Engineering
6. Bivariate Analysis
7. Data Transformation
8. Skewness Analysis
9. Data Engineering Validation
10. Machine Learning Readiness Assessment
11. Business Insights & Recommendations

---

## Key Findings

* Attrition Rate: 16.12%
* Overtime significantly increases attrition risk.
* Frequent business travel increases attrition.
* Promotion delays impact retention.
* Lower compensation correlates with higher attrition.
